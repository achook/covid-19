# covid-19

Small PWA to check number of persons affected by the new coronavirus

Uses amazing [API by mathdroid](https://github.com/mathdroid/covid-19-api)

Hosted on Google Firebase

## Project setup
``npm install``

### Development server
``npm run serve``

### Compile and minify for production
``npm run build``

### Lint and fix files
``npm run lint``

### Build and deploy
``npm run deploy``
